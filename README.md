## Features
  * Mudah dijalankan
  * Game Profile Info ( Fortnite, RainbowSix, dan osu! )
  * Memiliki Command yang Simple dan bersih
  * Info tentang Anime/Manga
  * Image Commands ( RIP, Pat, Neko, dan lainnya! )
  * Stay Safe with Corona Info!
